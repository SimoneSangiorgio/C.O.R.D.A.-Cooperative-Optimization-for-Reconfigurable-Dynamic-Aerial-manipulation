# C.O.R.D.A.: Cooperative Optimization for Reconfigurable Dynamic Aerial-manipulation
Development of a framework for the coordinated control of multi-UAV systems used for cable-based payload transport. The main contribution is
a real-time optimizer that dynamically adjusts the geometry of the UAVs formation,
parameterized by the cone angle, elliptical deformation and attitude to satisfy the
6-DOF wrench requirements of the payload while minimizing energy waste and
maximizing stability of the system.
